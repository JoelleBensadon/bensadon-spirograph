### Bensadon-Spirograph

### Spirograph

This project is a Java Swing application that creates a spirograph based on values 
entered by the user. The user can adjust the large radius, small radius, pen distance, 
number of steps, and angle per step, and then press Draw to see the design. The spirograph
is drawn by calculating points in a loop and connecting them with lines. The project is 
organized using MVC to separate the logic, data, and UI.
